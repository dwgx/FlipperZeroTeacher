# FlipperZero 中文入口

这里是中文主入口。建议优先从这里进入整个仓库，而不是直接从零散文件开始。

## 你现在应该点哪里

### 如果你是零基础读者

先进入 [中文教学导航](Guide/README.md)，然后按编号顺序从 [00. 学习总图](Guide/00-Study-Map.md) 开始往下读。

### 如果你已经准备开始开发 App

先进入 [03. App 开发](Guide/03-App-Development.md)，然后继续阅读 [04. 构建、调试与工具链](Guide/04-Build-Debug-Tools.md)、[07. 文件格式](Guide/07-File-Formats.md)、[06. 系统编程](Guide/06-System-Programming.md)。

### 如果你主要想整理资料或给 AI 使用

先进入 [中文主知识库](FlipperZero-Master-CN.md)，再阅读 [官方文档中文精读总表](Official-Docs-CN-Full.md) 和 [中文 AI 规则包](FlipperZero-AI-Pack-CN.md)。
