## 🎯 如何使用

| 你的情况 | 推荐路径 |
|:---|:---|
| 第一次接触仓库 | `00` → `01` → `02` → ... |
| 会基本使用设备 | 至少读 `00` + `01`，再根据目标选 |
| 想开发 App | 不要跳过 `02` + `03` |
| 给 AI 整理知识 | 配合主知识库、AI规则包一起使用 |
