## Goal

Understand the Flipper Zero JavaScript path and its boundaries.

## Concepts you must understand

- JavaScript is the lightweight scripting path
- it is not Node.js and not a browser runtime
- the JS SDK and the device runtime are different layers
- module loading exists under memory constraints

## Expected outcome

- decide whether a task fits the JS route
- distinguish script-layer capability from system-layer capability
