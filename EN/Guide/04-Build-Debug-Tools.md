# 04. Build, Debug, and Tools

## Goal

Understand the official build chain, IDE workflow, Dev Board flow, and update mechanisms.

## Core pages

- `https://developer.flipper.net/flipperzero/doxygen/dev_tools.html`
- `https://developer.flipper.net/flipperzero/doxygen/fbt.html`
- `https://developer.flipper.net/flipperzero/doxygen/vscode.html`
- `https://developer.flipper.net/flipperzero/doxygen/dev_board.html`
- `https://developer.flipper.net/flipperzero/doxygen/ota_updates.html`

## Concepts you must understand

- the division between `FBT` and `uFBT`
- `build/latest` and `compile_commands.json`
- VS Code workspace generation
- USB, Wi-Fi, logging, and debug modes on the Dev Board
- the high-level OTA update flow

## Expected outcome

- build a clean mental model of the official toolchain
- know when to use `FBT` and when `uFBT` is enough
- know where to start when debugging toolchain issues
