## 3. Core Reading Layers

- `Guide/`
  用途：顺序化章节学习

- `Master`
  用途：主线知识库

- `Official Docs Full`
  用途：官方文档深度整理

- `AI Pack`
  用途：检索、RAG 和模型规则

- `Official-Docs-Coverage.md`
  用途：追踪官方资料覆盖度和未完成项

## 4. Directory Policy

- `CN/` 和 `EN/` 是当前主结构
- `Original-Notes/` 保留原始资料
- `Legacy/` 只保留早期混合参考文件，不作为主入口
