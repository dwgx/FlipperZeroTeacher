# Wiki Resources

这个目录用于保存以后可直接放入 GitHub 的 Flipper Zero 官方资料归档。

## 目录结构

- `Official-Web-Snapshots/`
  保存官网网页快照，优先用于以后离线查看和应急保底。

- `Official-Source-Repos/`
  保存官方公开源码仓库中与文档直接相关的内容，优先用于以后整理 Markdown、图片和构建资源。

## 说明

- 这批内容的用途是长期保存、整理与应急查阅。
- 网页快照更适合“以后站点没了还能看”。
- 源码仓库更适合“以后继续整理、转换、重建 wiki 页面”。
- 官方站点一旦还在线，仍应优先以官方站点为准。

## 公开仓库注意事项

- 我目前没有找到一份非常明确的官方公开许可，来说明你可以把整站镜像长期公开再分发。
- 更稳妥的做法是：
  - 私有保存完整快照
  - 公开仓库中保留你自己的整理稿、索引、摘要和必要引用
  - 如果要长期公开整站镜像，最好确认官方许可或保留明确来源说明
