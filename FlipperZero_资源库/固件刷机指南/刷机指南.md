## 方法1: 使用 qFlipper 桌面应用 (推荐)

### 步骤

1. **下载 qFlipper**
   - Windows/Mac/Linux 通用
   - 官网: https://flipperzero.one/update

2. **连接 Flipper Zero**
   - 使用USB-C数据线连接电脑
   - 打开 qFlipper 应用
   - 等待识别设备

3. **安装固件**
   - 点击 "Install from file"
   - 选择下载的 `.tgz` 固件文件
   - 等待刷机完成

### 固件下载地址

#### Momentum Firmware
```bash
# 最新版本下载
https://github.com/Next-Flip/Momentum-Firmware/releases/latest

# 文件名格式
momentum-firmware-vX.X.X.tgz
```

#### Unleashed Firmware
```bash
# 最新版本下载
https://github.com/DarkFlippers/unleashed-firmware/releases/latest

# 文件名格式
flipper-z-f7-update-unleashed-vX.X.X.tgz
```

## 方法2: SD卡离线刷机

### 步骤

1. **下载固件**
   ```bash
   # 下载到电脑
   wget https://github.com/.../firmware.tgz
   ```

2. **解压到SD卡**
   ```
   SD卡根目录/
   └── update/
       └── momentum-firmware-vX.X.X/
           ├── flipper-z-f7-update-momentum-vX.X.X.tgz
           ├── resources.tar
           └── ...
   ```

3. **插入SD卡刷机**
   - 将SD卡插入 Flipper Zero
   - 打开设备，进入: `Settings` → `Update`
   - 选择固件文件确认更新
   - 等待自动重启

## 方法3: 命令行刷机 (开发者)

### 环境准备

#### macOS
```bash
# 安装依赖
brew install dfu-util protobuf nanopb

# 克隆源码
git clone --recursive https://github.com/Next-Flip/Momentum-Firmware.git
cd Momentum-Firmware
```

#### Linux (Ubuntu/Debian)
```bash
# 安装依赖
sudo apt-get update
sudo apt-get install -y \
    git \
    python3 \
    python3-pip \
    protobuf-compiler \
    nanopb \
    dfu-util

# 克隆源码
git clone --recursive https://github.com/Next-Flip/Momentum-Firmware.git
cd Momentum-Firmware
```

### 编译固件
```bash
# 编译完整固件
./fbt

# 编译更新包
./fbt updater_package

# 编译完成后，更新包位置:
dist/f7-C/f7-update-momentum/
```

### 刷入固件
```bash
# 方式1: 通过USB直接刷入 (设备需进入DFU模式)
./fbt flash_usb

# 方式2: 使用dfu-util手动刷入
dfu-util -d 0483:df11 -a 0 -s 0x08000000:leave -D firmware.bin

# 方式3: 生成更新包后通过SD卡安装
./fbt updater_package
cp -r dist/f7-C/f7-update-momentum /Volumes/FLIPPER/update/
```

## DFU 模式进入方法

### 方法A: 按键组合
1. 关机状态下按住 **BACK** 键
2. 插入USB数据线
3. 等待屏幕显示 "DFU Mode"

### 方法B: 软件进入
```bash
# 通过串口命令进入DFU
# 连接串口后发送:
dfu
```

## 固件切换

### 双系统切换 (Momentum特有)
Momentum固件支持保留官方固件:
```
Settings → System → Firmware Update → Switch to Stock
```

### 回退官方固件
1. 下载官方固件
2. 通过qFlipper或SD卡刷入
3. 清除SD卡上自定义应用

## 常见问题

### Q: 刷机失败/变砖怎么办?
**A:** 使用DFU模式强制刷机
```bash
# 进入DFU模式后
dfu-util -d 0483:df11 -a 0 -s 0x08000000:leave -D firmware.bin
```

### Q: 刷机后无法开机?
**A:**
1. 取出SD卡再开机
2. 检查SD卡文件系统是否为FAT32
3. 重新格式化SD卡并重刷

### Q: 如何备份原固件?
**A:**
```bash
# 读取固件到文件
dfu-util -d 0483:df11 -a 0 -s 0x08000000:307200 -U backup.bin
```

### Q: 刷机后设置丢失?
**A:** 固件更新会保留设置，但建议刷机前备份:
```
SD卡/backup/ 目录下的配置文件
```

## 刷机后配置

### 必做设置
1. **区域设置** - Settings → Region
2. **SubGHz频率解锁** - Settings → SubGHz (Unleashed/Momentum)
3. **开启Debug** - Settings → System → Debug
4. **设置时间** - Settings → Date & Time

### 安装应用
```
Momentum固件:
- 按向上键打开应用商店
- 或访问 https://flipc.org
```
